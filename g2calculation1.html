Add single digit numbers and double digit numbers

What is addition?
Addition means putting numbers together to find the total. We use the + sign to show addition, and the answer is called the sum.
How to add single digit numbers:
  • Start with the first number and put it in your head.
  • Count on the amount of times mentioned in the 2nd number.
  • For example: 5 + 3, start with 5 then count on 3 more times so 5, 6, 7, 8. The answer is 8!
How to add double digit numbers (without carrying):
  • Add the tens digit first: if you’re adding 23 + 12, start with the tens: 20 + 10 = 30.
  • Add the ones digit: 3 + 2 = 5.
  • Put them together: 30 + 5 = 35
How to add double digit numbers (with carrying):
  • Add the ones first: for 37 + 25, start with the ones: 7 + 5 = 12.
  • Carry over to the tens: write down the 2 in the ones place and carry the 1 to the tens column.
  • Add the tens: 30 + 20 = 50, don’t forget to add the 1 you carried over! 50 + 1 = 51.
  • Put it together: 37 + 25 = 62.
