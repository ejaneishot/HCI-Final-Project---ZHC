Identifying Positions on a Grid

What is a Grid?
A grid is a pattern of squares made by crossing horizontal (across) and vertical (up-and-down) lines. Itâ€™s like a giant checkerboard with numbered rows and columns. This is also known as a Cartesian plane.

How to Identify Positions on a Grid
The grid is divided into rows (horizontal lines) and columns (vertical lines). Each row and column is labeled with numbers or letters.

Example: The rows might be numbered 1, 2, 3, and the columns might be labeled A, B, C.

Reading Coordinates
A position on a grid is written like this: (Column, Row).

Example: (B, 3) means go to column B and row 3 to find the position.

How to Find a Position
First, find the correct column (horizontal letter). Then find the correct row (vertical number).

Example: If the position is (C, 2), go to column C and row 2 to locate it.
