Understand Halves, Quarters, and Fraction Equivalents

What is a fraction?
Fractions are parts of a whole. When we split something into equal parts, each part is called a fraction.
Halves (1/2):
When we cut something into 2 equal parts, each part is called a half.
Examples:
  • If you cut a pizza into 2 equal slices, one slice is a half.
  • If we share 8 candies equally between 2 people, each gets 4. That’s half!
→
Quarters (1/4):
When we cut something into 4 equal parts, each part is called a quarter.
Examples:
  • If you cut an apple into 4 pieces, one piece is a quarter.
  • If we share 12 cookies equally between 4 people, each gets 3. That’s a quarter!
→
Fraction Equivalents:
Sometimes fractions look different but mean the same thing. This is because when we divide the bottom number with the top number, it still makes the same result.
Examples:
  • Two quarters (2/4) are the same as one half (1/2)
  • This is because if we cut a pizza into 4 slices and take 2, it’s the same as taking 1 big half.
