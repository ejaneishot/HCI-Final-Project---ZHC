Ways to Sort and Organize Data
Block Graphs and Pictograms:
Block graph: A block graph uses blocks to show how many of something there is. Each block represents one item or group.
Pictogram: A pictogram uses pictures or symbols to show data, with each picture representing a certain number of items.

Lists and Tables
List: A list is a way to write things one after another to keep track of them.
Table: A table organizes information into rows and columns to make it easy to compare.

Example:
A list of how many animals we see at the zoo:
Lions: 3
Monkeys: 5
Tigers: 2

Which animal did we see most of in the zoo? Monkeys!
Venn Diagram
A Venn diagram is a picture with overlapping circles that shows how things are grouped. The overlapping part shows what two groups have in common.
